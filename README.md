# Bonus-project
<br>
Что делает приложение: Создает матрицу приоритетов бизнес-процессов, заодно позволяя нам просматривать список этих процессов и, при необходимости, изменять их (Добавить, удалить, вернуть к стандратным значеням). Процессам автоматически назначают соответствующий приоритет.
<br>
Как запустить: Клонируйте репозиторий в удобную вам папку, откройте приложение через файл index.html
